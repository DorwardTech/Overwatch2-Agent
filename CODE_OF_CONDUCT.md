# Code of Conduct

## Our pledge

We want participation in this project to be a friendly and welcoming experience
for everyone. We expect all contributors, maintainers, and participants to act
professionally and respectfully in every project space — issues, pull requests,
discussions, and any other communication.

## Our standard

This project adopts the **[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)**
as its code of conduct. In short: be respectful and considerate, assume good
faith, welcome newcomers, give and accept constructive feedback gracefully, and
focus on what is best for the community. Disruptive or unwelcome conduct is not
acceptable.

Please read the full text here:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Scope

This Code of Conduct applies in all project spaces, and also when an individual
is officially representing the project in public spaces.

## Reporting

If you experience or witness unacceptable behaviour, please report it privately
to the project maintainer, **[@DorwardTech](https://github.com/DorwardTech)**.
Reports will be reviewed and handled discreetly, and the privacy of the reporter
will be respected.
